## Desenvolvimento das Features de PTO [Requests and Approvals]

### Solicitações de PTO

- [ ] Criar o resource de `PtoRequests`.
- [ ] Implementar o resource de `PtoRequests` com os padrões da aplicação,
    utilizando a pasta `Shared` para itens em comum.
- [ ] Implementar o `RelationManager` de `PtoRequests` dentro do resource de `Users`, relacionando com `CompanyUser`.
- [ ] Implementar o `RelationManager` de `PtoRequests` dentro do resource de `Companies`, para visualizar todas as solicitações à empresa.

### Aprovações de PTO

- [ ] Criar o resource de `PtoApprovals`.
- [ ] Implementar o resource de `PtoApprovals` com os padrões da aplicação,
    utilizando a pasta `Shared` para itens em comum.
- [ ] Implementar o `RelationManager` de `PtoApprovals` dentro do resource de `PtoRequests`,
    permitindo que quem possua permissão aprove ou rejeite solicitações.
- [ ] Implementar o `RelationManager` de `PtoApprovals` dentro do resource de `Users`, relacionando com `CompanyUser`.
