## Desenvolvimento das Features de Relatórios

### Relatórios

- [ ] Criar o resource de `ReportJobs`.
- [ ] Implementar o resource de `ReportJobs` com os padrões da aplicação,
    utilizando a pasta `Shared` para itens em comum.
- [ ] Permitir a criação dos relatórios baseado nas permissões do usuário logado.
- [ ] Implementar filtros para os relatórios, como data, tipo de relatório e status.
- [ ] Implementar a geração de relatórios em segundo plano utilizando jobs.
- [ ] Implementar notificações para os usuários quando os relatórios estiverem prontos para download.
- [ ] Implementar o download dos relatórios em formatos como PDF e Excel.
