## Desenvolvimento das Features de Perfil

### Perfil

- [ ] Criar o resource de `User`, mostrando apenas o prório perfil.
- [ ] Implementar o resource de `User` com os padrões da aplicação,
  utilizando a pasta `Shared` para itens em comum.
- [ ] Implementar o `RelationManager` de `ResourceProfile` dentro do resource de `User`.
- [ ] Implementar o `RelationManager` de `Skills` dentro do resource de `User` como readonly.
- [ ] Implementar o `RelationManager` de `Roles` dentro do resource de `User` como readonly.
- [ ] Implementar o `RelationManager` de `PtoRequests` dentro do resource de `User`, mostrando o status
    baseado no `PtoApproval` relacionado.
- [ ] Permitir que os usuários editem apenas seu próprio perfil, conforme permissões.
- [ ] Mostrar o item de menu de `Profile` apenas para usuários autenticados.

### Alteração de Senha

- [ ] Implementar a funcionalidade de alteração de senha para o usuário logado.
- [ ] Garantir que a alteração de senha exija a senha atual para validação.
- [ ] Notificar o usuário sobre a alteração de senha bem-sucedida.
