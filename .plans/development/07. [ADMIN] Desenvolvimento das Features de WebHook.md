## Desenvolvimento das Features de WebHooks

### WebHooks

- [ ] Criar o resource de `WebHooks`.
- [ ] Implementar o resource de `WebHooks` com os padrões da aplicação,
    utilizando a pasta `Shared` para itens em comum.
- [ ] Implementar o `RelationManager` de `WebHooks` dentro do resource de `Companies`.
- [ ] Implementar a funcionalidade de disparo de WebHooks em eventos relevantes
    (ex.: criação de PTO, atualização de Tarefas, etc.).

### WebHook Deliveries

- [ ] Criar o resource de `WebHookDeliveries`.
- [ ] Implementar o resource de `WebHookDeliveries` com os padrões da aplicação,
    utilizando a pasta `Shared` para itens em comum.
- [ ] Implementar o `RelationManager` de `WebHookDeliveries` dentro do resource de `WebHooks`,
    permitindo visualizar o histórico de entregas de cada WebHook.
