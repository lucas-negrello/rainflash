## Desenvolvimento das Features de Projetos, Tarefas e Apontamento de Horas

### Projetos

- [ ] Criar o resource de `Projects`.
- [ ] Implementar o resource de `Projects` com os padrões da aplicação,
  utilizando a pasta `Shared` para itens em comum.
- [ ] Mostrar apenas os projetos da empresa do usuário logado, conforme permissões.
- [ ] Mostrar o item de menu de `Projects` apenas para usuários com permissão.

### Tarefas

- [ ] Criar o resource de `Tasks`.
- [ ] Implementar o resource de `Tasks` com os padrões da aplicação,
  utilizando a pasta `Shared` para itens em comum.
- [ ] Implementar o `RelationManager` de `ProjectTasks` dentro do resource de `Projects`.
- [ ] Mostrar apenas as tarefas dos projetos da empresa do usuário logado, conforme permissões.
- [ ] Mostrar o item de menu de `Tasks` apenas para usuários com permissão.

### Atribuições

- [ ] Implementar o `RelationManager` de `Users` dentro do resource de `Projects` através de `Assignments`.
- [ ] Utilizar a pasta `Shared` para itens em comum.
- [ ] Garantir que apenas usuários atribuídos possam ver e interagir com as tarefas e projetos.

### Apontamento de Horas

- [ ] Implementar o `RelationManager` de `TimeEntries` dentro dos resources de `Tasks`, e `Projects`
  através de `TimeEntries`.
- [ ] Utilizar a pasta `Shared` para itens em comum.
- [ ] Garantir que os usuários possam registrar horas apenas para tarefas e projetos aos quais estão atribuídos.
